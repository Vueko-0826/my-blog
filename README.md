# my-blog

这是我的个人主页项目，用于展示我的研究兴趣、项目经历、论文信息以及简历 PDF。

在线访问地址：

- 主页：[https://vueko-0826.github.io/my-blog/](https://vueko-0826.github.io/my-blog/)
- 简历 PDF：[https://vueko-0826.github.io/my-blog/resume.pdf](https://vueko-0826.github.io/my-blog/resume.pdf)

## 项目内容

- 个人主页首页
- 研究兴趣与项目经历展示
- 论文与文章板块
- 简历 PDF 链接

## 文件结构

```text
my-blog/

├── style.css
├── script.js

├── .gitignore
└── README.md
```

## 本地查看

直接在浏览器中打开 `index.html` 即可。

如果使用 VS Code，可以在项目目录执行：

```powershell
code .
```

## 更新网站

修改文件后，在项目目录执行：

```powershell
git status
git add .
git commit -m "写清楚这次更新内容"
git push
```

推送成功后，GitHub Pages 会自动更新网站内容。
